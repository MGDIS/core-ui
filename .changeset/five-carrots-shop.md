[**`<mg-illustrated-message>`**](http://core.pages.mgdis.fr/core-ui/core-ui/storybook/?path=/docs/molecules-mg-illustrated-message--docs) updates illustrated message sizing to set both horizontal and vertical directions, aligns the `title` typography with the heading font, and adds `style="fill: var(--mg-b-color-app)"` while keeping the original `fill` and `.dynamic-color` class to get SVG illustration theming by default.
