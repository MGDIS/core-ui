[**`<mg-illustrated-message>`**](http://core.pages.mgdis.fr/core-ui/core-ui/storybook/?path=/docs/molecules-mg-illustrated-message--docs) updates illustrated message sizing to set both horizontal and vertical directions, aligns the `title` typography with the heading font, and supports SVG illustration theming via the `.dynamic-color` class (fill uses `--mg-b-color-app`).
