Layout

Grid

You can use grid classes by using the .mg-l-grid class in the container. This call will add flex properties to the container.

Auto layout

Default column class is mg-l-grid__col. The width will be defined by the number of element using the class.

Sized column

Like Bootstrap, you can use sized colomn based on a 12 colomns grid.

The library is using the BEM methodology, so to define a column width we need to set a modifier class, for exemple mg-l-grid__col-2.
